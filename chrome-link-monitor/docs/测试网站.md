# 测试网站清单（除招标网外的通用站点）

扩展 **不仅** 限于安徽省招标网、合肥公共资源。只有这两个域名走「门户模式」；**其余所有 http(s) 网站** 走通用模式：后台 `fetch` 页面（带 Cookie），失败时再读已打开的标签页。

---

## 一、内置测试（最省事）

扩展弹窗 → **打开测试页**，或地址：

`chrome-extension://<你的扩展ID>/test/monitor-test.html`

在扩展管理页 `chrome://extensions` 可看到扩展 ID。

---

## 二、推荐公网测试地址

### 测「有变化」— 间隔设 5 秒

| 网站 | URL | 说明 |
|------|-----|------|
| **httpbingo UUID**（首选） | `https://httpbingo.org/uuid` | 每次返回新 UUID，**第二次检查必变** |
| random.org 随机数 | `https://www.random.org/integers/?num=10&min=1&max=100&col=10&base=10&format=html&rnd=new` | 每次 HTML 数字不同 |

> **不要用** `httpbin.org`：服务已不稳定，扩展会自动改写到 httpbingo，请直接用 httpbingo。

### 测「无变化」— 验证不误报

| 网站 | URL | 说明 |
|------|-----|------|
| Example | `https://example.com` | 经典静态页，多次检查应「正文无变化」 |
| httpbingo 固定 HTML | `https://httpbingo.org/html` | 固定 HTML 片段 |
| W3C 示例 | `https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf` | ⚠️ PDF 不适合测正文，仅作「打不开 HTML」参考 |

### 测「真实网站」— 手动验证通用能力

| 网站 | 怎么测 | 预期 |
|------|--------|------|
| **GitHub 仓库** | 打开 `https://github.com/某用户/某仓库`，右键或 `lj` 监控当前页 | 能加入列表；Star 数变化可能触发提醒（属正常） |
| **维基百科** | 打开任意文章页，监控当前标签页 | 能检查；内容通常较稳定 |
| **百度 / 新闻站** | 监控某篇 **详情页** URL，不要监控首页 | 首页轮播、广告易误报 |
| **政府站其他省份** | 非 ahtba/ggzy 的 `.gov.cn` 详情页 | 与 GitHub 一样走通用 fetch，**无右下角工具条** |

## 三、两种模式对比

| | 门户站（ahtba、ggzy） | 其他所有网站 |
|--|----------------------|--------------|
| 域名 | `ahtba.org.cn`、`ggzy.hefei.gov.cn` | 任意其他域名 |
| 添加方式 | 点选模式、本页链接、工具条 | 右键、`lj`、面板输入 URL |
| 读内容方式 | 优先匹配的标签页注入脚本 | 后台 fetch → 可选已开标签 |
| 页面 UI | 右下角工具条 | 无（可选「可监控」仅门户有） |
| 自动开标签 | 受「选择链接后自动打开」开关控制 | 一般不自动开（除非打开该开关且逻辑命中门户 URL） |

## 四、标准测试流程

### A. 证明「变了能提醒」（UUID）

1. 右键 `https://httpbingo.org/uuid` → 开启链接监控  
2. 间隔 **5 秒**，保存，开始监控  
3. 等 10 秒或点两次「立即检查」  
4. 应有强提醒（第二次相对第一次内容变）  

### B. 证明「没变不吵」（Example）

1. 打开 `https://example.com`  
2. 空白处右键 → 监控当前标签页网址，或地址栏 `lj` 回车  
3. 点三次「立即检查」  
4. 均应提示正文无变化  

### C. 证明「普通站右键可用」

1. 打开 GitHub 任意公开仓库  
2. 在「Issues」或「Code」链接上右键 → 开启链接监控  
3. 面板「正在监控」应出现对应 URL

## 五、常见问题

| 现象 | 原因 | 处理 |
|------|------|------|
| httpbin 失败 | 服务废弃 | 改用 **httpbingo.org/uuid** |
| 普通站没工具条 | 设计如此 | 用右键 / `lj` / 面板 |
| 需要登录的站检查失败 | fetch 无登录态 | 浏览器登录后 **保持该页标签打开** |
| 首页老误报 | 列表/广告在变 | 改监控 **详情页** URL |
| PDF 链接监控无效 | 扩展只比 HTML 指纹 | 换 HTML 详情页 |

## 六、自造本地测试页（可选）

若公网不稳定，可用项目内测试页上的链接，或本地起一个静态服务：

```bash
# 在 chrome-link-monitor/test 目录
python -m http.server 8765
```

浏览器打开 `http://localhost:8765/monitor-test.html`，右键页面内链接即可。
