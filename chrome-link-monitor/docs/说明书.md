# 链接变更监控 — 用户使用说明书

**版本：2.0.16**  
**适用浏览器：Google Chrome（Manifest V3 扩展）**

---

## 一、这款扩展是做什么的？

「链接变更监控」帮你在浏览器里**盯住指定网页**：按你设定的时间间隔自动检查页面内容有没有变，一旦正文发生变化，会通过 **Windows 系统通知、Chrome 弹窗、提示音** 等方式提醒你。

典型用途：

- 招投标公告详情页：中标、流标、答疑有没有更新  
- 新闻、商品详情、论坛帖子等需要「盯着看」的页面  
- 用测试页验证提醒功能是否正常  

**它不会**理解「状态=成交」这类语义，只判断页面 HTML/文本指纹是否变化。

## 二、安装

### 2.1 加载扩展

1. 打开 Chrome，地址栏输入 `chrome://extensions/`  
2. 右上角打开 **开发者模式**  
3. 点击 **加载已解压的扩展程序**  
4. 选择文件夹 **`chrome-link-monitor`**（文件夹内应直接能看到 `manifest.json`）  
5. **不要**选上一级目录，否则扩展无法工作  

### 2.2 允许通知

- Windows：**设置 → 系统 → 通知** → 允许 **Google Chrome** 发通知  
- 否则只能看到扩展面板里的文字提示，没有右下角系统通知  

### 2.3 确认安装成功

- 扩展列表中显示「链接变更监控」，版本 **2.0.16**  
- 工具栏出现扩展图标  
- 「检查视图」里 Service Worker 显示 **（无效）** 是正常现象，表示后台休眠，不是损坏（详见文末 FAQ）

## 三、界面总览（扩展弹窗）

点击工具栏图标打开设置面板，主要区域如下：

| 区域 | 作用 |
|------|------|
| **添加监控链接** | 输入 URL、名称，设置检查间隔，点「开始监控」 |
| **正在监控** | 当前列表，可打开页面或单独停止 |
| **快捷监控** | 监控当前标签页 / 最近选中的链接 |
| **本页链接（招标网）** | 扫描门户站可监控链接；可开关「可监控」标记 |
| **弹窗提醒** | 噪声过滤、弹窗节流相关选项 |
| **启动行为** | 浏览器启动、选择链接后是否自动打开页面 |
| **底部按钮** | 测试页、立即检查、测试通知、停止全部 |

## 四、添加监控的多种方式

### 4.1 扩展面板手动添加（通用）

1. 在「添加监控链接」输入框填入 **完整网址**（可填名称方便辨认）  
2. 点 **+ 添加链接** 可增加多行  
3. 设置检查间隔（默认 30 分钟；测试可用 5 秒）  
4. 点 **开始监控**  

首次开始会建立基准并检查一次；若内容无变化，**不会**弹强提醒。

### 4.2 右键菜单（任意网站）

| 操作位置 | 菜单项 |
|----------|--------|
| **链接上**右键 | 开启链接监控 |
| **页面空白处**右键 | 监控当前标签页网址 |
| **扩展图标上**右键 | 监控当前标签页网址 |
| **选中网址文字**后右键 | 监控选中的网址 |

关闭监控：链接右键「关闭链接监控」，或面板里停止单条 / 停止全部。

> **注意：** 地址栏（网址搜索栏）本身**不能**挂扩展右键菜单，这是 Chrome 限制。

### 4.3 地址栏快捷命令 `lj`

1. 打开要监控的页面  
2. 点击地址栏，输入 **`lj`**（进入扩展命令模式）  
3. **直接按 Enter** → 监控当前标签页网址  
4. 或输入 **`lj github.com/xxx`** 再 Enter → 监控指定网址  

### 4.4 安徽省招标网 / 合肥公共资源（门户站）

适配域名：

- `www.ahtba.org.cn` / `ahtba.org.cn`  
- `ggzy.hefei.gov.cn` 及子域  

这些站点常 **禁用链接右键**，请用下面方式：

#### 方式 A：点选模式（推荐）

1. 打开列表页或首页  
2. 页面右下角工具条 → **开启点选模式**  
3. **点击**要监控的列表条目（点选会拦截跳转，并自动加入监控）  
4. 约 **3 秒** 后自动退出点选；右上角也可 **关闭点选模式**  
5. 默认使用全局检查间隔（通常 30 分钟）  

#### 方式 B：本页链接列表

1. 打开扩展弹窗 → **本页链接（招标网）**  
2. 点 **刷新链接列表**  
3. 点击列表中某一条即可添加监控  

#### 方式 C：工具条按钮

- **监控本页**：监控当前打开的页面 URL  
- **监控已选项**：监控点选模式下选中的链接  

#### 「可监控」标记（可选）

- 弹窗中勾选「在页面上显示『可监控』标记」  
- 可监控链接旁会出现红色小标签，**仅提示、不可点击**  
- 默认 **关闭**，避免页面杂乱

## 五、重要设置说明

### 5.1 检查间隔

- 可勾选「所有链接使用相同检查间隔」，或为每条链接单独设置  
- 快捷预设：5 秒、5 分、30 分、1 时、1 天、1 周  
- 改间隔后点 **保存检测间隔**；已开始监控的会按新间隔调度  

**建议：** 政府招投标门户 **≥ 30 分钟**；功能测试用内置 uuid 页 **5 秒**。

### 5.2 选择链接后自动打开监控页面

| 状态 | 行为 |
|------|------|
| **关闭（默认）** | 添加监控后 **不** 新建标签页打开该链接 |
| **打开** | 添加监控后在后台自动打开监控页面标签 |

适用于点选模式、本页链接、右键添加等所有添加路径。

关闭时，定时检查会尽量用 **已有标签页** 或 **HTTP** 读取；招标网若需登录，请 **手动打开详情页并保持标签不关**。

### 5.3 打开浏览器时自动打开监控页面

- 与上一项不同：控制 **每次启动 Chrome** 时是否自动打开正在监控的页面  
- 首次开始监控时可能会询问是否开启  

### 5.4 忽略纯广告/弹窗变化

- **开启**：只有正文变化才强提醒；仅广告、浮层变化时静默或轻量通知  
- **关闭**：装饰性变化也会轻量系统通知  

### 5.5 弹窗相关

- **60 分钟内弹窗超过 40 次时询问是否减缓**：防止变更频繁时弹窗刷屏  
- **暂停 Chrome 弹窗与网页遮罩**：仍保留 Windows 系统通知  
- **测试通知** 不计入 40 次限额

## 六、提醒方式说明

| 情况 | 你会看到什么 |
|------|----------------|
| **正文内容变化** | 系统通知 + Chrome 弹窗 + 网页遮罩 + 三声提示音 + 扩展角标 |
| **仅广告/浮层变化** | 轻量系统通知（标题会注明可能为广告/弹窗） |
| **勾选忽略噪声** | 上述轻量情况也可能被静默 |
| **内容无变化** | 无提醒（面板「立即检查」会提示「正文无变化」） |
| **首次建立基准** | 无强提醒 |
| **检查失败** | 系统通知（同一 URL 15 分钟内失败通知会降噪） |

## 七、测试扩展是否正常

### 7.1 测试通知渠道

扩展弹窗 → **测试通知**  
应看到系统通知并听到提示音（需手动关闭该测试通知）。

### 7.2 测试变更检测

1. 扩展弹窗 → **打开测试页**  
2. 在测试页点击 **httpbingo.org/uuid** 链接，或手动添加该 URL  
3. 间隔设为 **5 秒**，开始监控  
4. 第二次检查时应收到 **内容已变更** 提醒

## 八、使用建议与场景

### 8.1 招投标 / 公告（推荐做法）

**适合：**

- 单个项目 **详情页** 链接  
- 页面上能直接看到状态文字  

**不建议：**

- 只监控 **首页列表**（其他条目更新、轮播、统计易导致误报）  
- 监控 PDF 附件内文字  
- 间隔过短轰炸网站  

**推荐流程（ahtba）：**

1. 打开首页或列表页  
2. 点选模式 → 点 **某条公告条目**  
3. 间隔 30 分钟或按网站更新频率调整  
4. 保持浏览器登录；需要时手动打开详情标签  

### 8.2 普通网站

1. 打开 **详情页**  
2. 右键链接或空白处 → 开启监控  
3. 需登录的站：登录后 **保持该标签页打开**

## 九、更新扩展后必做

在 `chrome://extensions` **重新加载**扩展后：

1. 所有已打开的招标网 / 监控相关页面 **F5 或 Ctrl+Shift+R**  
2. 右下角工具条应显示 **链接变更监控 v2.0.16**  
3. 扩展卡片 → **错误 → 全部清除**（清除旧版残留报错）  
4. 指纹逻辑更新后，建议 **删除误报条目并重新点选**  

若不刷新页面，可能出现点选无反应、`Extension context invalidated` 等。

## 十、常见问题（FAQ）

| 问题 | 解决办法 |
|------|----------|
| Service Worker 显示「无效」 | 正常休眠，有操作时会自动唤醒 |
| 招标网没有工具条 | 确认域名；F5 刷新；看版本号 |
| 点选后没加入监控 | 重载扩展 + 刷新页面 |
| 没变化也报变更 | 删除监控，重新点选 **详情链接**；勿只监控首页 |
| 变了却不提醒 | 变更在 PDF/纯 JS 层；打开详情标签并保持 |
| 检查失败 / 503 / 504 | 加大间隔；保持标签页打开；稍后重试 |
| 完全没有系统通知 | Windows 允许 Chrome 通知 |
| 弹窗太多 | 暂停弹窗；加大间隔；勾选忽略噪声 |
| 地址栏不能右键监控 | Chrome 限制；用 `lj` 命令或页面右键 |

## 十一、能力边界（做不到的事）

- 不解析语义，只比对页面指纹  
- 不监控附件（PDF、Word 等）内文字  
- 纯 Canvas / WebSocket 且不落 DOM 的变更可能漏报  
- 极短间隔在 Chrome 后台休眠时可能不准  
- 请遵守目标网站服务条款，合理设置间隔
