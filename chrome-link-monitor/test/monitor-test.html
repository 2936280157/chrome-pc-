链接变更监控 — 通用网站测试页

除安徽省招标网、合肥公共资源外，任意普通网站均可监控。 扩展在后台直接请求 URL（带 Cookie），无需专用工具条。

快速测「有变化」：右键下面 UUID 链接 → 开启链接监控 → 间隔设 5 秒 → 第二次检查应强提醒。
快速测「无变化」：监控 example.com 或下方静态页 → 立即检查应提示正文无变化。

① 推荐：UUID（每次内容都变）

使用 httpbingo.org（勿用已失效的 httpbin.org）。间隔 5 秒，第二次检查必报变更。

④ 其他普通网站怎么测

方式操作
链接右键 在任意网页的链接上右键 → 开启链接监控
当前页 页面空白处右键 → 监控当前标签页网址
地址栏 输入 lj 回车（监控当前标签页）
扩展面板 粘贴 URL → 开始监控；或「监控当前标签页」
GitHub 等 打开仓库/议题页 → 右键或 lj，与招标网无关

标准测试步骤（UUID）

  1. 右键上方 UUID 链接 → 开启链接监控
  2. 扩展面板：间隔 5 秒，保存，确认已开始监控
  3. 等待约 10 秒，或点两次「立即检查」
  4. 应出现：系统通知 + Chrome 弹窗 + 提示音（首次仅建基准，第二次才提醒）
